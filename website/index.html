Built for companionship

不打扰工作，
但一直都在。

从桌面窗口到宠物包管理，每个细节都围绕“轻量、安静、可控”设计。 默认不联网，不需要账号，也不会把一个可爱桌宠做成复杂的控制中心。

01 · Desktop native

真正住在桌面上的小伙伴

透明无边框、跳过任务栏、可置顶，也能从托盘快速锁定、隐藏或退出。

02 · Cross platform

三个系统，体验一致

Windows、macOS、Linux 都有原生安装包，配置和宠物包结构保持统一。

WINMACLINUX
03 · Your collection

宠物来源不受限

扫描 Codex 默认目录，也能添加任意本地文件夹，随时切换自己的角色收藏。

04 · Optional AI

想聊时再接入模型

AI 对话默认关闭。填入 OpenAI 兼容接口后，桌宠才会出现聊天入口。

05 · Hi-DPI ready

放大也尽量清晰

根据窗口尺寸和屏幕倍率自动选择 1×、2× 或 4× 图集。

1× · 2× · 4×

Download

选好系统，
让桌面多一位伙伴。

安装后从托盘打开设置，即可导入图鉴中的桌宠或本地宠物包。 所有构建都可在 GitHub Releases 中核对与下载。

推荐
Windows 10 / 11

Windows

下载 setup.exe 双击安装；企业或静默部署场景可选 MSI。

下载 setup.exe MSI 备用 未签名版本可能触发 SmartScreen 提示。
Apple Silicon / Intel

macOS

按芯片选择 DMG，下载后拖入 Applications 文件夹。

Apple Silicon Intel Mac 未公证构建需在“隐私与安全性”中允许。
AppImage / deb / rpm

Linux

AppImage 适合直接试用，Debian、Fedora 系发行版也有原生包。

下载 AppImage
deb rpm
首次运行 AppImage 前需添加执行权限。
