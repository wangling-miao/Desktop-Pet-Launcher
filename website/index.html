Launcher

桌宠体验，装进一个轻量启动器。

01

真正像桌宠

透明无边框、跳过任务栏、可置顶，也能在托盘里显示、隐藏、锁定和退出。

02

位置和大小可控

拖动后保存位置，设置页可调整宽高、动画速度、点击穿透和开机自启。

03

宠物来源不受限

默认扫描 Codex 宠物目录，也可以添加自己的本地文件夹作为宠物包库。

04

高清素材优先

根据窗口尺寸和屏幕倍率选择 1x、2x 或 4x atlas，放大后尽量保持清晰。

Codex compatible

兼容现有 Codex 桌宠，也给高清新包留足空间。

旧版 hatch-pet 包可以继续使用：pet.json、spritesheet.webp、192 x 208 cell、 1536 x 1872 atlas 都保持兼容。新的 desktop-pet skill 会优先制作 4x master， 再导出 2x 和 1x，让桌宠不是从小图硬拉大。

Download

下载最新版，开始放一只桌宠在屏幕边上。

选择你的系统，下载推荐安装包。安装完成后从托盘打开设置， 就能导入桌宠图鉴或本地宠物包。

Windows

推荐 setup.exe

适合大多数 Windows 10/11 用户，双击安装即可。企业或静默部署场景可选 MSI。

下载 Windows 安装器
